# 🎋 竹斬り道場 - Ultimate Bamboo Cutting

## 概要
究極の竹斬り体験を提供するWebゲームです。リアルタイム物理演算、没入感のある効果音、そして爽快な斬撃アクションが特徴です。

## 🎮 ゲーム特徴

### ✨ 主要機能
- **🎯 精密斬撃システム**: マウス/タッチによる直感的な斬撃操作
- **⚡ リアルタイム物理演算**: 竹の落下、回転、重力シミュレーション
- **🎵 Web Audio API音響**: 手動生成による高品質効果音
- **💥 パーティクルエフェクト**: 斬撃時の迫力あるビジュアル
- **🏆 コンボシステム**: 連続斬撃でハイスコア獲得

### 🎌 難易度レベル
- 🌱 **新竹 (Easy)**: 初心者向け、体力1、速度遅め
- 🎋 **普通竹 (Normal)**: 標準設定、体力2、標準速度
- 🎍 **古竹 (Hard)**: 上級者向け、体力3、高速
- ⚔️ **鉄竹 (Extreme)**: 達人向け、体力5、超高速

### 🧘 特別モード
- **禅モード**: 心を落ち着ける瞑想的体験
- **エピックモード**: 最大パフォーマンスでの超爽快体験

## 🎯 ゲームルール

### 基本操作
1. 🖱️ **マウス操作**: 竹を一直線にドラッグして斬撃
2. 📱 **タッチ操作**: 画面をスワイプして斬撃
3. ⚡ **連続斬撃**: 2秒以内の連続カットでコンボ発生
4. 💎 **パーフェクトカット**: 竹の中心を狙うとボーナス

### スコアシステム
- **切断数**: 基本得点 (×10点)
- **最高コンボ**: ボーナス得点 (×50点)
- **パーフェクトカット**: 特別音響効果
- **ハイスコア**: ローカルストレージに自動保存

## 🔧 技術仕様

### アーキテクチャ
```
┌─ HTML5 Canvas (3層構成) ─┐
│ ├─ gameCanvas (竹描画)    │
│ ├─ particleCanvas (粒子)  │ 
│ └─ effectCanvas (エフェクト)│
└─────────────────────────┘
┌─ Web Audio API ─────────┐
│ ├─ 動的音響生成          │
│ ├─ フィルター処理        │
│ └─ リアルタイム再生      │
└─────────────────────────┘
```

### 物理エンジン
- **重力シミュレーション**: リアルな落下物理
- **回転演算**: 3D回転エフェクト
- **衝突判定**: 線分と矩形の高精度交差判定
- **パーティクル物理**: 爆発時の粒子挙動

### 音響システム
```javascript
// 斬撃音生成例
const oscillator = audioContext.createOscillator();
const filter = audioContext.createBiquadFilter();
oscillator.type = 'sawtooth';
filter.type = 'highpass';
filter.frequency.value = 1000;
```
